class QtRestClient::MetaComponent

template<typename T, typename = void>
class QtRestClient::MetaComponent< T, typename >

A helper class to perform operations on either a QObject or a Q_GADGET.

Template Parameters
TThe type to be checked for QObject/Q_GADGET

The specializations all define the following methods:

  • deleteLater
  • deleteAllLater

The component can be used to check if a type fullfills the requirements to be used with the rest client, and perform deletion operations, if possible.

◆ removeGlobalApi()

void QtRestClient::removeGlobalApi ( const QString &  name,
bool  deleteClient = true 
)

Removes a previously added API from the global list.

Parameters
nameThe name of the API to be removed.
deleteClientSpecifies whether the removed client should be deleted

If you don't delete the client, It will be automatically deleted together with the QCoreApplication. If you don't want this, simply retake ownership by yourself:

auto client = QtRestClient::apiClient("name");
QtRestClient::removeGlobalApi("name");
client->setParent(this);
